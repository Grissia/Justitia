## Usage

### Docker

最快的方式
可以直接使用下列指令

```bash
docker compose up --build -d
```

所有日研工具就會直接上線了

### 老方法

如果用老方法就不需要那堆 docker 的東東
總之就是下載下來
確定你的 python 有裝 flask
然後執行下列指令

```bash
python app.py
// 或是
python3 app.py
```

他就會自己飛出來啦
